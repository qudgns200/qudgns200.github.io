PC-picks는 당신의 데스크탑, 노트북 등 PC를 자랑하기 위해 만든 사이트 입니다. 사용자의 PC 사진을 제목, 설명과 함께 업로드 할 수 있습니다.

🛠 본 사이트는 개인의 개발 역량을 키우기 위해 만들어지고 있는 중이며 게시판 기초를 다시 한 번 다지기 위해 제작하였습니다. 사용한 기술 스택은 아래와 같습니다.

## 🌱 Front-end

**개발 언어**
JAVASCRIPT, HTML, CSS

## 🌱 Back-end

**개발언어**
JAVA

**프레임워크**
SPRING BOOT

**데이터베이스**
MYSQL


## 🔥 구현 진척도

 - [O] 로그인/회원가입
 - [O] 글 올리기
 - [O] 메인페이지
 - [O] 글 상세페이지
 - [O] 페이징
